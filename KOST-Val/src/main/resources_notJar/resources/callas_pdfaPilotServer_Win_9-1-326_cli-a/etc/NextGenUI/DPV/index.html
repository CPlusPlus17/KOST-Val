To select multiple nodes, use ⌘/Ctrl+Left Click

To open all selected nodes completely, use ⌘/Ctrl+Shift+o

To close all selected nodes completely, use ⌘/Ctrl+Shift+c

To navigate inside the treeview, use ↑↓→←

To jump to the first node, use ⌘/Ctrl+Page Up or Home

To jump to the last node, use ⌘/Ctrl+Page Down or End

To copy the selected node and its open children, use ⌘/Ctrl+c
